# Bloom Botanical Atelier 🌿

> **"Petals of Soft Silence"** — A digital sanctuary for curated botanical artifacts and floral narratives.

Bloom is a premium, minimalist e-commerce experience designed for a high-end botanical brand. Moving away from clinical grids and rigid templates, this project embraces **"The Botanical Atelier"** design system — prioritizing organic asymmetry, airy white space, and a soft, rhythmic editorial flow.

## 🎨 The Design Philosophy

The aesthetic is built on the **"Tonal Layering"** principle. Instead of using harsh borders and dividing lines, Bloom uses subtle Background Color Shifts to define hierarchy.

*   **Palette**: Organic tones (Dusty Rose, Sage Green, Sun-drenched Yellow) set against a "Bright and Airy" surface (#FAF9F6).
*   **Typography**: A tension between the refined **Noto Serif** (Editorial Voice) and the geometric **Plus Jakarta Sans** (Functional UI).
*   **Aesthetics**: Glassmorphism navigation, ultra-diffused shadows, and large 3rem radii for a "soft" brand personality.

## 📁 Project Structure

The codebase is structured to be lightweight and performant, utilizing Vanilla JS and Tailwind CSS (CDN) for rapid, bespoke styling.

```text
├── index.html           # The Homepage (Hero & Heroine of the site)
├── catalog.html         # Curated product listing with faceted filters
├── product.html         # Product details featuring an asymmetric gallery
├── cart.html            # Minimalist shopping cart & selection summary
├── about.html           # The "Our Story" narrative
├── journal.html         # Editorial blog for slow-living insights
├── sustainability.html  # Transparency report & environmental impact
├── css/
│   └── style.css        # Design tokens & global utility classes
└── js/
    └── main.js          # Cart logic & shared UI interactions
```

## 🛠️ Tech Stack

*   **Core**: HTML5, Vanilla JavaScript.
*   **Styling**: Tailwind CSS (Utility-first framework).
*   **Fonts**: Google Fonts (Noto Serif & Plus Jakarta Sans).
*   **Icons**: Material Symbols Outlined (Custom light-weight settings).

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to enter the Atelier. No build step or installation required.
